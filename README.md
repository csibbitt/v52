# Virtual 52 

An internet deck of cards. Play free-form or with rulesets.

PRE-ALPHA project that barely works and might be dangerous.

Also, none of the contributors have written for node.js before.

## Installation

1. Install node.js
1. Clone v52 repo

## Usage

1. node v52Server.js
